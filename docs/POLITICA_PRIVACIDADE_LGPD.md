# Politica De Privacidade E LGPD

## Dados Tratados

O sistema pode tratar nome do solicitante, contato, origem, destino, descricao da demanda, responsavel, status, prazos, notas e historico operacional.

## Finalidade

Os dados sao usados para organizar entregas, coletas, tarefas, prazos, responsaveis, comunicacao operacional, auditoria e relatorios.

## Acesso

O painel administrativo exige login. A consulta publica exige protocolo e contato e nao mostra notas internas.

## Retencao

Os dados devem ser mantidos pelo periodo necessario para operacao, auditoria, suporte, obrigacoes legais ou politica interna do cliente.

## Direitos Do Titular

Solicitantes podem pedir correcao, consulta ou exclusao de dados conforme processo definido pelo controlador.

## Seguranca

O sistema aplica validacao de entradas, sessao assinada, cookie `HttpOnly`, login administrativo, exportacao controlada e auditoria.

## Observacao

Este documento e uma base tecnica-operacional e nao substitui revisao juridica.
